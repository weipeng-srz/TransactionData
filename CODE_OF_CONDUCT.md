# 社区行为准则

我们希望 TickLens 社区保持开放、友善、专业，并欢迎不同背景与经验水平的参与者。

## 我们鼓励的行为

- 围绕事实、代码和可复现结果讨论问题；
- 尊重不同观点，给出具体、建设性的反馈；
- 主动说明假设、数据口径和不确定性；
- 在发现错误时及时更正，并帮助新贡献者理解项目约定。

## 不可接受的行为

- 骚扰、歧视、侮辱、威胁或人身攻击；
- 未经允许公开他人的个人信息、账号或交易数据；
- 刷屏、恶意营销、操纵市场式宣传或收益承诺；
- 鼓励绕过访问控制、数据授权、法律法规或第三方服务条款；
- 其他明显破坏协作环境的行为。

## 执行

请通过 GitHub 向仓库维护者 `@weipeng-srz` 报告行为问题。维护者可以编辑或删除不当内容，限制参与者互动，或在必要时禁止参与。报告会在合理范围内保密处理。

本准则适用于仓库、Issue、Pull Request、Discussion 以及代表项目参与的其他公开空间。
