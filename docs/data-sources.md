# 数据来源、限制与合规边界

TickLens 使用公开可访问的数据接口，不附带任何第三方历史行情、新闻或财务数据，也不提供第三方服务的可用性保证。

## 数据来源

| 类别 | 当前用途 |
| --- | --- |
| 新浪财经公开数据 | 前复权因子、实时/历史行情和新闻检索 |
| 东方财富公开数据 | 证券简称、估值、分红和财务报表 |
| Cloudflare D1 | 研究状态和匿名聚合遥测 |

浏览器只访问同域 `/api/*` 路由；具体上游接口由 `web/app/lib` 中的服务端客户端模块封装，可能随数据提供方调整而变化。

## 已知限制

- 免费接口可能限流、要求验证、改变字段或临时不可用。
- 历史行情是日 K 聚合数据，不包含逐笔委托、撤单和订单链路。
- 新闻搜索结果可能遗漏、重复、延迟或相关性不足。
- 财务规则和技术信号是研究辅助，不能检测所有会计问题，也不能预测收益。
- 名称检索和主要行情研究范围以沪深 A 股为主。

## 使用边界

- 遵守数据提供方条款、robots 规则、访问频率限制和所在地法律法规。
- 不要使用本项目绕过登录、验证码、访问控制、付费墙或授权限制。
- 对外分发抓取结果前，确认你拥有存储和再分发这些数据的权利。
- 商业、生产或高频场景应接入有明确授权、稳定性承诺和数据质量责任的数据服务。

## 隐私

- 浏览器存储可包含最近查询、标注和视图偏好；共享设备上请主动清理。
- 托管环境以邮箱的 SHA-256 派生值作为用户键，不把邮箱原文写入 D1。
- 匿名遥测只记录白名单事件的每日总次数和总耗时，不记录查询股票或研究内容。

## 投资风险

所有“主力行为”、B/S 指引、新闻情绪、财务异常和回测结果都是模型或规则代理。它们可能受到样本偏差、未来函数、复权口径、数据缺失、交易成本和市场制度变化影响。使用者应自行验证并承担决策风险。
