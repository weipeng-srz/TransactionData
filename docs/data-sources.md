# 数据来源、限制与合规边界

TickLens 使用公开可访问的数据接口和行情协议。它不附带任何第三方历史行情、新闻或财务数据，也不提供第三方服务的可用性保证。

## 数据来源

| 类别 | 当前用途 |
| --- | --- |
| 通达信免费行情协议 | Go 命令的沪深 A 股 Level-1 历史分笔 |
| 新浪财经公开数据 | 前复权因子、实时/历史行情和部分新闻检索 |
| 东方财富公开数据 | 证券简称、股本结构、估值、分红和财务报表 |
| 中国新闻网公开搜索 | 新闻检索入口之一 |

具体接口由对应客户端模块封装，可能随上游调整而变化。

## 已知限制

- 免费接口可能限流、要求验证、改变字段或临时不可用。
- 历史分笔不是交易所 Level-2 数据，不含委托、撤单和订单链路。
- 新闻搜索结果可能遗漏、重复、延迟或相关性不足。
- 财务规则和技术信号是研究辅助，不能检测所有会计问题，也不能预测收益。
- 当前历史分笔可靠支持沪深市场；北交所代码会被明确拒绝。
- Web 的名称检索和主要行情研究范围以沪深 A 股为主。

## 使用边界

- 遵守数据提供方条款、robots 规则、访问频率限制和所在地法律法规。
- 不要使用本项目绕过登录、验证码、访问控制、付费墙或授权限制。
- 对外分发抓取结果前，确认你拥有存储和再分发这些数据的权利。
- 商业、生产或高频场景应接入有明确授权、稳定性承诺和数据质量责任的数据服务。

## 隐私

- CLI 只把结果写到用户指定的本地路径。
- Web 浏览器存储可包含最近查询、自选股、标注和视图偏好；共享设备上请主动清理。
- 托管环境以邮箱的 SHA-256 派生值作为用户键，不把邮箱原文写入 D1。
- 匿名遥测只记录白名单事件的每日总次数和总耗时，不记录查询股票或研究内容。

## 投资风险

所有“主力行为”、B/S 指引、新闻情绪、财务异常和回测结果都是模型或规则代理。它们可能受到样本偏差、未来函数、复权口径、数据缺失、交易成本和市场制度变化影响。使用者应自行验证并承担决策风险。
